# ☕ Java Learning Repository

This repository documents my journey learning Java programming through a structured online course (Bro Code) and additional practice projects.

The goal of this project is to build a strong foundation in Java, object-oriented programming, and software development principles as part of my preparation for an internship or apprenticeship (e.g., at SAP).

---

## What I learned

This repository covers a wide range of Java topics:

### Fundamentals
- Variables & Data Types
- User Input
- Operators & Expressions
- If Statements & Switch
- Loops (for, while, nested loops)
- Strings & Methods

### Object-Oriented Programming (OOP)
- Classes & Objects
- Constructors
- Inheritance
- Polymorphism
- Abstraction & Interfaces
- Encapsulation (Getters & Setters)

### Data Structures
- Arrays & 2D Arrays
- ArrayLists
- HashMaps
- Enums
- Wrapper Classes

### Advanced Topics
- File Handling (Read/Write)
- Exception Handling
- Generics
- Multithreading
- Dates & Timers
