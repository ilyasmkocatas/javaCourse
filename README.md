# ☕ Java Course Learning Portfolio

A structured Java learning repository built while completing a long-form Java course. This project documents my progress from core Java fundamentals to object-oriented programming, collections, file handling, and multi-threading.

The goal of this repository is not only to store course exercises, but to present them in a clean, recruiter-friendly format that shows technical growth, code organization, and hands-on practice.

## Why this repository exists

* To document my Java learning journey in a structured way
* To organize course exercises by topic instead of leaving them as an unstructured dump
* To highlight selected mini-projects that demonstrate practical programming ability
* To prepare a professional portfolio for future applications, including dual-study programs in Wirtschaftsinformatik

## Repository structure

```text
javaCourse/
├── fundamentals/
├── control-flow/
├── methods/
├── oop/
├── data-structures/
├── advanced/
├── mini-projects/
└── docs/
```

### Topic folders

* **fundamentals** – variables, input, arithmetic, strings, formatting
* **control-flow** – if statements, loops, switch, logical operators
* **methods** – method creation, overloading, varargs, scope
* **oop** – classes, constructors, inheritance, polymorphism, abstraction, interfaces, composition
* **data-structures** – arrays, 2D arrays, ArrayLists, HashMaps, enums
* **advanced** – exception handling, file I/O, generics, dates, threading

## How to run

Most examples are standard Java files and can be run from a Java IDE such as BlueJ, IntelliJ IDEA, or Eclipse.

If you want to run a specific class manually from the terminal, compile and execute it with your local Java installation:

```bash
javac YourClassName.java
java YourClassName
```

> Note: Some exercises were originally created in BlueJ, so class names and structure may reflect that workflow.
